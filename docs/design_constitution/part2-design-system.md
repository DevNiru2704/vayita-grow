# MASTER AWWWARDS WEBSITE PROMPT

## Part 2 - Visual Design System

> This section defines how the AI should construct a cohesive visual design system. It should derive the system from the project itself rather than applying predefined aesthetics.

---

# 1. Visual Design Philosophy

A professional website is not a collection of attractive sections.

It is a **single visual language** repeated consistently across every page.

Every visual decision should feel like it belongs to the same family.

The website should appear as though it was designed by one highly experienced designer-not by multiple unrelated contributors.

---

# 2. Design System First

Before creating any component, establish a complete design system.

Determine:

- Typography
- Color Palette
- Grid
- Layout Widths
- Spacing Scale
- Border Radius
- Shadow System
- Icon Style
- Illustration Style
- Button Language
- Card Language
- Motion Language
- Image Treatment

No component should exist outside this system.

---

The AI must infer the correct visual direction from the project.

Examples:

- Clean

---

## Gaming

---

---

# 4. Typography System

Typography defines personality more than color.

Before selecting fonts determine:

- Formal or informal?
- Editorial or technical?
- Luxury or corporate?
- Human or geometric?
- Playful or serious?

---

## Typography Hierarchy

Create a complete hierarchy.

Example:

- Display
- Hero Heading
- Section Heading
- H2
- H3
- H4
- Body Large
- Body
- Small
- Caption
- Labels
- Buttons
- Navigation

Every size should belong to a consistent scale.

---

## Font Pairing

Choose fonts intentionally.

Examples:

Editorial

- Serif + Sans

Corporate

- Modern Sans + Sans

Creative

- Experimental Display + Neutral Sans

Technical

- Geometric Sans + Monospace Accent

Luxury

- Elegant Serif + Minimal Sans

Never combine fonts randomly.

---

## Typography Rules

Maintain:

- predictable line height
- consistent letter spacing
- readable paragraph width
- sufficient contrast
- proper rhythm
- consistent capitalization

Avoid excessive font weights.

---

Determine:

Secondary

Accent

Border

Success

Danger

Muted

## Color Philosophy

Never add colors because they are trendy.

Instead ask:

Does this color reinforce the brand?

Does it improve readability?

Does it increase hierarchy?

Does it strengthen accessibility?

---

## Dark vs Light

Choose based on:

Audience

Content

Brand identity

Industry

Lighting conditions

Device usage

Do not default to dark mode.

---

# 6. Spacing System

Spacing creates perceived quality.

Use a mathematical spacing scale.

Example:

4

8

12

16

24

32

48

64

96

128

160

192

Avoid arbitrary spacing values.

---

## White Space

Allow content to breathe.

Premium interfaces often use **more** whitespace rather than less.

Never compress layouts unnecessarily.

---

# 7. Layout System

Determine:

Maximum width

Content width

Grid columns

Container padding

Responsive margins

Vertical rhythm

Section spacing

Maintain consistency across every page.

---

## Grid Philosophy

Grid should organize-not restrict.

Use grids to establish harmony.

Break the grid only when doing so improves storytelling.

# 17. Visual Hierarchy

Hierarchy should be obvious without reading.

Guide attention using:

Scale

Contrast

Spacing

Alignment

Position

Typography

Motion

Never rely on color alone.

---

# 18. Consistency Rules

Once a design decision is made, repeat it consistently.

Examples:

If cards have 20px radius,

every related card should use the same radius.

If buttons animate in 200ms,

all buttons should behave similarly.

If headings use a specific spacing ratio,

maintain it throughout the site.

Consistency builds trust.

---

# 19. Responsive Philosophy

Do not merely shrink layouts.

Redesign them.

Consider:

Touch targets

Reading width

Thumb reach

Spacing

Typography scale

Navigation

Images

Motion

Each breakpoint should feel intentionally designed.

---

# 20. Premium Details

Professional websites pay attention to details.

Examples:

Perfect alignment

Optical spacing

Balanced typography

Consistent rhythm

Thoughtful empty space

High-quality imagery

Unified shadows

Predictable interactions

Subtle polish

These small details collectively create a premium experience.

---

# 21. Visual System Checklist

Before implementation, verify:

- Does every component belong to the same design system?
- Is typography consistent?
- Is spacing mathematical?
- Are colors purposeful?
- Is hierarchy clear?
- Are layouts balanced?
- Are icons unified?
- Are illustrations stylistically consistent?
- Is responsiveness intentional?
- Would a user describe the interface as cohesive?

If any answer is "no," refine the design system before proceeding.
